# Golang Backend Boilerplate

# Run Migration 
```go
go run main.go migrate
```

# Run Locally
Install dependencies

```go
go mod vendor
```

### Start the server Locally

```go
go run main.go serve
```

# Start the server using Docker

```makefile
make development
```
# Stop the server using Docker

```makefile
make clean
```
